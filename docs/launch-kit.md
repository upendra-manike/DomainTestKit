# Launch Kit

Use this kit to announce DomainTestKit and attract early adopters.

## Positioning Statement

DomainTestKit is an enterprise Playwright framework for teams that need more than UI automation: domain flows, API tests, accessibility checks, performance baselines, security hooks, and CI-ready reporting in one structure.

## LinkedIn Post Draft

We just open-sourced **DomainTestKit**: an enterprise Playwright automation framework built for **FinTech, Healthcare, and E-commerce** teams.

Most test frameworks stop at UI checks. DomainTestKit includes:
- UI + API testing
- domain business flows
- accessibility and performance checks
- security baseline hooks
- CI/Docker-ready execution
- Playwright + Allure reporting

If your team is scaling automation across domains, this repo is designed to be a practical starter.
